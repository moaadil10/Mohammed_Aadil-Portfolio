01. Professional Summary

Software Engineer with 2+ years of experience building production-grade backend systems and RESTful APIs using C#, ASP.NET Web API, ASP.NET MVC, ASP.NET Core, Entity Framework, and ADO.NET.

Currently working at Benzy Infotech on WebConnect Plus, a Central Reservation System (CRS) and airline B2B portal platform supporting 10 regional markets across GCC, UK, Kenya, Tanzania, and other international regions. My work involves developing and maintaining APIs for flight availability search, booking workflows, and airline provider integrations using XML and JSON APIs.

I work on provider-based architecture using design patterns such as Facade, Factory, Adapter, and Strategy to support multiple airline providers and scalable integrations. I implement asynchronous programming using Async/Await and Task.WhenAll for parallel provider calls and optimize performance using Redis and Aerospike caching with the cache-aside pattern. I also work on structured logging with correlation IDs, MySQL query optimization, stored procedures, transactions, and production support across multiple regional portals.

In addition to my professional work, I build projects independently to strengthen my full-stack and architecture skills. My projects include an Inventory and Billing System built using ASP.NET Core, Angular, MySQL, Entity Framework Core, JWT, Swagger, and xUnit; an AES Encryption Tool built using ASP.NET Core MVC with SHA-256 key derivation and SOLID principles; and an E-Commerce application built using ASP.NET WebForms, ADO.NET, and SQL Server with stored procedures and authentication.
